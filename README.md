# DocuVerify AI

A web-based document OCR and risk-based verification system using
Django, Django REST Framework, Docker, and AWS.

## Features
- Aadhaar & PAN OCR
- Format & consistency checks
- Risk scoring engine
- Web dashboard
- REST APIs

## Tech Stack
- Django, DRF
- PostgreSQL
- OpenCV, Tesseract
- Docker
- AWS
